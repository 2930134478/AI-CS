# 如何添加界面截图

## 📸 步骤说明

### 1. 准备截图
截取您想要展示的界面，建议使用以下方式：
- 使用浏览器开发者工具（F12）截取完整页面
- 使用截图工具（如 Snipaste、Windows 截图工具等）
- 确保截图清晰、完整

### 2. 命名图片
将截图保存为以下文件名之一（支持 PNG 或 JPG 格式）：

| 界面 | 文件名 |
|------|--------|
| 工作台 | `dashboard.png` 或 `dashboard.jpg` |
| 访客端 | `visitor.png` 或 `visitor.jpg` |
| AI配置 | `ai-config.png` 或 `ai-config.jpg` |
| 用户管理 | `users.png` 或 `users.jpg` |
| FAQ管理 | `faq.png` 或 `faq.jpg` |

### 3. 放置文件
将截图文件复制到以下目录：
```
frontend/public/images/screenshots/
```

### 4. 查看效果
刷新网页，打开界面展示部分，即可看到您的截图。

## 📋 图片要求

- **格式**：PNG 或 JPG
- **尺寸**：建议 1920x1080 或保持 16:9 比例
- **大小**：每张图片建议不超过 2MB，以保证加载速度
- **质量**：清晰、完整，能够展示界面核心功能

## 💡 提示

- 如果图片不存在或加载失败，系统会自动显示占位符图标
- 您可以随时替换截图文件，刷新页面即可看到更新
- 建议使用 PNG 格式以获得更好的质量
